Review of concepts

We are just going to do some very basic things first, some basic calculations. Part of R is like a glorified calculator and we can do some basic calculations. Type in the following, in turn, and see what comes back:

1+1
## [1] 2

Say now we want to save the result of this calculation. We can put it into a variable called Result (check the environment window. You‚Äôll now see the variable ‚ÄúResult‚Äù listed).

Let‚Äôs say we want to create a new vector, going from 1 to 10. We can use the c function in R (combine).

But we can do this a bit easier too (combine, from 1 to 10).

Lastly, say we want to get one value, or one line, from our variable Dependent. We can do this quite easily too. To get the first value of the first column/row we say:

## [1] 3

To get the first row we say Dependent[1,].
