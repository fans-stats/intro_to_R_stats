Background:

The data set consists of health information on a sample of n = 45 small US cities. The following measures have been recorded, by city:

  • death: death rate per 1000 residents
  • doctor: doctor availability per 100,000 residents
  • hospital: hospital availability per 100,000 residents
  • income: annual per capita income in thousands of dollars
  • density: population density people per square mile

  1. The error terms 𝜀 are normally distributed
dataset %>%
  lm(formula = doctor ~ hospital + income) %>%
  augment() %>%

  ggplot(aes(x = .std.resid)) +
    geom_histogram(binwidth = 0.5) +
    theme_bw()

The residuals 𝜖ϵ are normally distributed. We can see in the histogram that the distribution is bell-shaped and symmetrical around the mean.

  1. The error terms have the same variance irrespective of the size of the predicted outcome (i.e., variance does not depend on x1 or x2), also called “homoscedasticity”.

We will build the scatterplot of standardized residuals and standardized predicted values.

dataset %>%

The scatterplot of standardized residuals and standardized predicted values shows no pattern.

Task 5

Provided the three assumptions to carry out statistical inference were met for the model derived in Task 3, consider testing some hypotheses next:

  1. Write the null and the alternative hypotheses for the regression coefficient  and report on the results.

    𝛽1

    β1

𝐻0H0: 𝛽1β1 coefficient equals 0 𝐻𝛼Hα: 𝛽1β1 coefficient is different than 0

We can infer the value of 𝛽1β1 = 0.100 for the whole population, as the 𝑝p value is less than 0.05 (𝑝p < 0.001). Therefore, we reject the null hypothesis of no association between the two variables and we infer that the number of available doctors is associated with the number of hospitals. We estimate that a 1 hospital increase is associated with a 0.1 doctor availability increase, for a given per capita income.

  1. Write the null and the alternative hypotheses for the regression coefficient  and considering that the model assumptions for inference are met, report on the results.

    𝛽2

    β2

𝐻0H0: 𝛽2β2 coefficient equals 0 𝐻𝛼Hα: 𝛽2β2 coefficient is different than 0

We can infer the value of 𝛽2β2 = 13.273 for the whole population, as the 𝑝p value is less than 0.05 (𝑝p = 0.011). Therefore, we reject the null hypothesis of no association between the two variables and we infer that the number of available doctors is associated with the income per capita. We estimate that a 1 thousand dollars increase in income per capita between cities is associated with an increase of 13.273 doctors per 100,000 population, for a given hospital availability.

  1. Report and interpret the 95% confidence interval for .

    𝛽1

    β1

The 95% confidence interval for 𝛽1β1 is [0.051, 0.148]. The estimated value for 𝛽1β1 in the 95% of samples extracted from the population will belong to this interval. This interval excludes the null value of 0, also telling us that the b1 parameter is significant, i.e. there is a significant relationship between hospital and doctor availability.

  1. Report and interpret the 95% confidence interval for .

Therefore, 41.8% of the variance in the number of available doctors is “explained” by the annual per capita income and hospital availability. 𝑅2R2 adjusted is 0.39, saying 39% of the variance in the number of available doctors is “explained” by the annual per capita income and hospital availability. This value should be used rather than the unadjusted 𝑅2R2 as it adjusts for the number of predictors, and is better to use when we are comparing models with different numbers of predictors.
