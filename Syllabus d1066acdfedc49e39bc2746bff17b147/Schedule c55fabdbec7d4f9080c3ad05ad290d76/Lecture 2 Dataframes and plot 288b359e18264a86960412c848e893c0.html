Okay finally we are going to do some basic plotting. As discussed in the lecture, we are going to use ggplot to plot, which is a wonderful tool to build beautiful and informative plots. These are easy to amend and would take a whole 2-3 hour session to go through but let‚Äôs give it a go!

First, let‚Äôs do a quick histogram of values. This could be used to determine whether data are normally distributed (other ways, like levenes test, shapiro wilkes etc, also exist). So we are going to make a ggplot with data = tutorial_data.

In the aesthetics we are going to make the value of interest GABA, and we are going to separate it by group (fill = Group) and then we are going to make a histogram. To make it a little prettier, we make it a little transparent (alpha = 0.6) and we make the binwidth = 0.5 but try different values and see how it affects the histogram!

ggplot(data = tutorial_data,
       aes(x=GABA,
           fill = Group)) +


  geom_histogram(alpha=0.6,
                 binwidth = 0.5)

Next, we are going to make boxplots of sensory values by group.

ggplot(data = tutorial_data,
       aes(x = Group,
           y = Sensory1,
           fill = Group)) +

  geom_boxplot()

This looks nice, but how about we add some better labels and a title

ggplot(data = tutorial_data,
       aes(y = Sensory1,
           x = Group,
           fill = Group)) +
  geom_boxplot() +
  theme_bw() +
  xlab("Group") +
  ylab("Sensory Scores") +
  ggtitle("Sensory Scores by group") +
  theme(plot.title = element_text(hjust = 0.5))#align title centerd

Here, we set a black and white theme (my preference) add x and y labels and a title, and center it.

Now we can also decide to add individual datapoints to this, to make it a little more transparent. So let‚Äôs do that using geom_point(). But we are going to do a few more things: a. We want individual datapoints to be the same as each boxplot they are in, so we are also going to color them by group b. We want a bit of a jitter to spread them out, so we use the position_jitterdodge function within geom_point() to jitter them and set some other parameters c.¬†We want all the individual points (so also outliers) to be colored, so in geom_boxplot we need to set it such that outliers are identified (outlier.shape = NA)

ggplot(data = tutorial_data,
       aes(y = Sensory1,
           x = Group,
           fill = Group)) +
  geom_boxplot(outlier.shape = NA) +
  geom_point(aes(y = Sensory1),
             position=position_jitterdodge(jitter.width = 0.4,
                                           jitter.height = 0,
                                           dodge.width = 1)) +

  theme_bw() +
  xlab("Group") +
  ylab("Sensory Scores") +
  ggtitle("Sensory Scores by group") +
  theme(plot.title = element_text(hjust = 0.5))#align title centered

We can also color them by Sex if we want to, in geom_point() we can set color to color = Sex. We can also play around with colors more but we don‚Äôt have time for this today!
